# Testes de Software
## Passos para executar o backend
### Configurar o .env de acordo com o .env-example

### Exemplo de configuração
```
SALT=10
SECRET=secret
NOME=admin
FUNCAO=admin
TELEFONE=(99) 99999-9999
EMAIL=admin@gmail.com
SENHA=admin
MAP_KEY='key-para-api-google-maps'
```
### Depois executar 
```
npm run build
npm start
```
